Analog Reading Project

Components Required

  • Arduino Uno
  • Potentionmeter
  • Breadboard
  • Jumper Wires

Description

This project makes an Analog reading project using Arduino. It is the basic project for beginners.

Arduino Code

/*
   Led blinking project 

   Components:
     - Arduino UNO 
     - Potentionmeter 
     - Breadboard 
     - Jumber wire
     Author: Circuit Creation RN

*/
 
#define pin A0

float read_value;
float voltage;

void setup() {
  Serial.begin(9600);
  pinMode(pin, INPUT);
}

void loop() {
  read_value = analogRead(pin);
  voltage = (read_value * 5.0) / 1023.0;
  Serial.println(voltage);
  delay(1000);
}
