LED Blinking Project

Components Required

  • Arduino Uno
  • LED
  • 1k Resistor
  • Breadboard
  • Jumper Wires

Description

This project makes an LED blink using Arduino. It is the first basic project for beginners.

Arduino Code

/*
   Led blinking project 

   Components:
     - Arduino UNO 
     - Led
     - 1k Resistor 
     - Breadboard 
     - Jumber wire
     Author: Circuit Creation RN

*/
 
int led = 13;

void setup(){
 pinMode(led, OUTPUT);
}

void loop(){
 digitalWrite(led, HIGH);
 delay(1000);
 digitalWrite(led, LOW);
 delay(1000);
}
