Arduino UNO LED Blinking Using Mobile Phone

Description

This beginner friendly Arduino project demonstrates how to blink an LED using an Arduino UNO without a laptop or PC. The Arduino code is written and uploaded using the Arduinodroid app on an Android phone with an OTG cable.

Components Required

  • Arduino UNO R3
  • LED
  • 220Ω Resistor (Recommended)
  • Arduino USB Cable
  • OTG Cable
  • Android Phone
  • Arduinodroid App

Circuit Connections

Step 1 – Connect the LED

Connect the long leg (anode) of the LED to Digital Pin 12 of the Arduino UNO through a 220Ω resistor.

Step 2 – Connect the LED Negative

Connect the short leg (cathode) of the LED to any GND (Ground) pin on the Arduino UNO.

Step 3 – Connect the Arduino

Connect the Arduino UNO to your Android phone using a USB cable and an OTG cable.

Step 4 – Open Arduinodroid

Open the Arduinodroid app, create a new sketch, and paste the LED blinking code.

Step 5 – Upload the Code

Select Arduino UNO as the board and tap the Upload button. After the upload is complete, the LED connected to Pin 12 will blink every second.

Circuit Explanation

The LED is connected to Digital Pin 12 of the Arduino UNO through a current-limiting resistor. The Arduino repeatedly turns the pin HIGH and LOW, making the LED blink at one-second intervals.

The program is uploaded directly from an Android phone using the Arduinodroid app and an OTG cable, eliminating the need for a laptop or desktop computer.

Code




/*
 Arduino LED Blink with Mobile Phone

 Components:
 - Arduino UNO R3
 - Arduino Cable
 - LED
 - OTG Cable

 Author: Circuit Creation RN
*/

int ledpin = 12;

void setup() {

  pinMode(ledpin, OUTPUT);

}

void loop() {

  digitalWrite(ledpin, HIGH);
  delay(1000);

  digitalWrite(ledpin, LOW);
  delay(1000);

}

Code Explanation

The setup() function runs once and configures Digital Pin 12 as an output. Inside the loop() function, the LED is turned ON for one second using digitalWrite(HIGH), then turned OFF for one second using digitalWrite(LOW). This process repeats continuously, creating a blinking LED effect.

Project Working

After uploading the program through the Arduinodroid app using an OTG cable, the Arduino UNO executes the code continuously. The LED connected to Digital Pin 12 blinks every second, demonstrating the basic operation of digital output control.

Learning Outcome

  • Learn LED blinking programming.
  • Upload Arduino code using Arduinodroid.
  • Program Arduino without a laptop or PC.

Common Mistakes

  • Wrong LED polarity.
  • Incorrect pin number in the code.
  • OTG cable not detected.
  • Board or COM port not selected correctly.
  • USB cable without data support.

FAQ

Can I upload Arduino code without a laptop?

Yes. You can upload code using the Arduinodroid app and an OTG cable with an Android phone.

Why is my LED not blinking?

Check the LED polarity, wiring, board selection, and ensure the code is uploaded successfully.

Which Arduino board is used?

This project uses the Arduino UNO R3.
