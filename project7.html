Touch Alarm Using BC547 Transistor

Components Required

  • 9V Battery
  • BC547 Transistor
  • 100k Resistor
  • Buzzer
  • Breadboard
  • Jumper Wires

Description

This Touch Alarm Project uses a BC547 transistor to detect touch signals and activate a buzzer instantly. It is a simple beginner electronics project that demonstrates how touch sensing works using basic components.
