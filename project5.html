LED CHASER Project

Components Required

  • 9V Battery
  • IC 555
  • IC CD4017
  • 10k Resistor
  • 220 ohms resistor
  • 10 mf Capacitor
  • LED
  • Breadboard
  • Jumper Wires

Description

This LED Chaser Circuit uses the IC 555 timer to generate clock pulses and the CD4017 decade counter to light up LEDs in sequence, creating a smooth running light effect. It’s a simple and fun beginner project to understand timing circuits and sequential logic using basic electronic components.
